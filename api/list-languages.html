.listLanguages()

更新日: 2018.09.14

listLanguages()

利用可能な言語名のリストを返します。


デモ

使用可能な言語を表示します。

JavaScript
var lang = hljs.listLanguages(),
    ul   = document.createElement('ul');
for (let i = 0; i < lang.length; i++) {
  let li = document.createElement('li');
  li.textContent = lang[i];
  ul.appendChild(li);
}
document.getElementById('lang').appendChild(ul);
HTML
<div id="lang"></div>
